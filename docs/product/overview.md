# 제품 개요

## TaskManager

유연한 일감 관리 시스템으로, 사용자가 커스텀 필드와 상태를 통해 자신만의 워크플로를 구성할 수 있습니다.

## 핵심 기능

### 1. 빠른 태스크 생성
- 사이드바 또는 모바일 하단 내비게이션의 "일감 생성" 버튼으로 모달에서 빠른 생성
- 제목, 유형, 상태, 프로젝트 선택 지원
- 기본 유형(태스크)과 기본 상태(열림)가 자동 할당

### 2. 유형 기반 일감 관리
- 유형(IssueType)별로 다른 필드와 상태 구성 가능
- 기본 유형: 태스크, 버그, 스토리

### 3. 커스텀 필드
- 어드민 화면에서 사용자가 직접 필드 생성 가능
- 필드 타입: 텍스트, 숫자, 날짜, 선택, 다중 선택, 사용자, URL
- 시스템 필드(제목, 설명, 담당자, 우선순위, 마감일)는 삭제 불가

### 4. 상태 관리
- 유형별 상태 워크플로 구성
- 상태 카테고리: TODO, IN_PROGRESS, DONE
- 시스템 상태(열림, 진행 중, 완료)는 삭제 불가

### 5. 프로젝트 관리
- 프로젝트 단위로 일감 분류 및 멤버 관리
- 인증 사용자 누구나 프로젝트 생성 가능
- 프로젝트 생성 시 생성자가 소유주이자 멤버로 자동 등록
- 소유주는 항상 최소 1명 필요하며, 프로젝트 단위에서 양도 가능
- 내가 멤버로 등록된 프로젝트만 사이드바/프로젝트 목록에 표시

### 6. 역할 기반 접근 제어
- 프로젝트별 역할(Role) 지정: 프로젝트 어드민, 관리자, 멤버, 뷰어(시스템 기본 제공)
- 프로젝트 소유주는 역할과 별개로 해당 프로젝트의 전체 권한을 가짐
- 권한 종류: `project:manage`, `members:manage`, `workitems:create/edit/delete/assign`, `comments:create/delete`
- 글로벌 ADMIN 계정은 전체 어드민 기능 접근 가능

### 7. 인라인 편집 및 다양한 뷰
- 리스트뷰, 그리드뷰, 칸반뷰, 간트뷰 4종 지원
- 상태·유형·제목 인라인 편집
- 칸반뷰: HTML5 드래그 앤 드롭으로 카드 이동 → 상태 자동 변경
- 그리드/간트 그리드: 컬럼 헤더 드래그로 가로 너비 조절 지원

## 지원 환경

- **웹**: 데스크톱 브라우저 (Chrome, Firefox, Safari, Edge)
- **모바일**: 반응형 디자인으로 모바일 브라우저 지원
- **테마**: 시스템 설정에 따른 다크/라이트 모드 자동 전환

## 날짜/시간 규약

- 날짜 표기: `yyyy-mm-dd(ddd)` 또는 `mm-dd(ddd)`
- 시간: 사용자 시스템 타임존 기준
